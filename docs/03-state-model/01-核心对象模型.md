# 01 核心对象模型

Hive 的最小核心对象集合（v0.2 对齐）：

- Idea
- Brief
- Plan
- Phase
- Task
- Decision
- Issue
- Handoff Record
- Checkpoint

## 建议写法

每个对象独立小节，固定结构：

1. 定义（Definition）
2. 最小字段（Required Fields）
3. 状态（Status）
4. 生命周期（Lifecycle）
5. 与其他对象关系（Relations）
