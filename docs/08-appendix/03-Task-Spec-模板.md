# 03 Task Spec 模板

## Purpose

- 提供标准 Task Spec 结构。
- 保证 Task 可派发、可验证、可恢复。

## Template

```md
# Task Spec

## Objective
- 

## Scope
- 

## Constraints
- 

## Plan References
- 

## Done Criteria
- 

## Validation Method
- 

## Output Format
- 

## Escalation Rule
- 
```

## Checklist

- Objective 明确
- Scope 有边界
- Done Criteria 可验证
- Validation Method 可执行
- Escalation Rule 明确
