# 05 Issue Record 模板

## Purpose

- 提供标准 Issue Record 结构。
- 保证失败或阻塞可分类、可恢复。

## Template

```md
# Issue Record

## Metadata
- Issue ID:
- Source Object:
- Severity:

## Failure Type
-

## Description
-

## Impact
-

## Evidence
-

## Suggested Actions
-

## Retry Strategy
-

## Escalation Path
-
```

## Checklist

- Failure Type 明确
- Description 可复现
- Impact 可判断
- Evidence 可回溯
- Suggested Actions 可执行
