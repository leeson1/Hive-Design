# 01 术语表

- User：目标提出者与运行中指令输入者
- Steering / Escalation Authority：高阶裁决职责，可由 human、policy engine 或 strategic planner 承担；旧文档中的 `Queen` 是其历史别名
- Orchestrator：运行时调度、派发、监控、验收与重规划层
- Drone：旧称，后续统一并入 Orchestrator
- Worker Agent：受约束的一次性执行层
- Directive：用户初始目标或运行中追加指令的结构化表示
- Research Sprint：有边界的调研单元
- Evidence Pack：调研结果的结构化证据载体
- Charter：稳定规则层
- Execution Plan：演进执行层
- Plan Revision：Execution Plan 的增量修订节点
- Task：可派发、可验证、可重试的工作单元
- Task Graph：由 Task 节点和依赖、冲突边构成的调度图
- AgentRun：某次 Task 派发对应的具体运行实例
- Handoff Record：Worker 退出前提交给 Orchestrator 的交接包
- Acceptance Record：Orchestrator 对 Handoff 的验收结果
- Artifact：代码、文档、报告、日志等产物引用
- Event：驱动状态迁移的不可变运行事实
- Issue：失败、阻塞、风险或待处理问题
- Checkpoint：恢复快照
