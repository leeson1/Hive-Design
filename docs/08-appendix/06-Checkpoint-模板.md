# 06 Checkpoint 模板

## Purpose

- 提供标准 Checkpoint 结构。
- 保证控制回合可恢复。

## Template

```md
# Checkpoint

## State Snapshot
-

## Active Directive
-

## Current Plan Revision
-

## Active AgentRuns
-

## Completed Tasks
-

## Open Tasks
-

## Open Issues
-

## Pending Decisions
-

## Blockers
-
```

## Checklist

- State Snapshot 够恢复
- Active Directive 清晰
- Active AgentRuns 清晰
- Completed Tasks 清晰
- Open Tasks 清晰
- Blockers 明确
