# 02 核心设计原则

## 原则 A：Deterministic State > Agent Memory

- 系统连续性必须来源于状态对象与检查点。

## 原则 B：Structured Tasks > Open-ended Instructions

- Worker 只接受结构化、可验证、可重试任务。

## 原则 C：Phase Transitions > Continuous Reasoning

- 以阶段门控推进，不依赖无限推理链。

## 原则 D：Recovery Checkpoints > Long Conversations

- 优先可恢复性，避免上下文丢失导致中断。
